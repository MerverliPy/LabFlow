# Handoff

Update `PHASE_HANDOFF.md` with:
- objective
- changed files
- blockers
- next exact step
- resume command
- proof status

Also update `STATE.md` with the current repo state.

## Rules
- keep the handoff short and specific
- include concrete filenames
- include exactly one recommended next action
