# Requirements

## Stable commands

- `init`
- `task`
- `session`
- `memory`
- `status`
- `doctor`

## Experimental

- providers
- security
- deploy
- codex

## Parked

- swarm
- neural
- learn
- giant agent catalogs

## Promotion rule

No experimental or parked area moves to stable without:

- owner
- tests
- docs
- rollback path
- proof artifact
- repeated real use
