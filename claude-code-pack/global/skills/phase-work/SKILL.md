# Phase Work

1. Read the project's active-truth files.
2. Restate the current phase objective briefly.
3. Execute only the current phase.
4. Prefer the smallest correct patch.
5. Run the smallest relevant verification.
6. Update `STATE.md`.
7. Update `PHASE_HANDOFF.md`.
8. Stop once complete or blocked.
